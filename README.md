# FinanceData
Finance data 

## ANBIMA YF.py

Este script usa a biblioteca yfinance para obter informações sobre o ativo PEJA11 e imprime essas informações na tela. Lembre-se de que os dados financeiros podem estar sujeitos a alterações, e é sempre bom verificar se há atualizações nas bibliotecas que você está utilizando.

## B3 B3.py

Neste exemplo, pandas_datareader é utilizado para obter os dados históricos do ativo PEJA11 da B3. Note que esses dados incluem informações sobre os preços de abertura, fechamento, máximo, mínimo, volume, etc., mas não incluem a taxa Anbima diretamente. Se a taxa Anbima for disponibilizada através da B3, você precisará procurar uma fonte específica ou verificar se ela está disponível nos dados retornados.
